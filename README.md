# 💪 AI Fitness & Workout Planner

An intelligent fitness web application built using **Streamlit + LangChain + Groq LLM** that generates personalized workout plans, diet charts, and fitness insights based on user inputs.

---

## 🚀 Features

### 🏋️ Personalized Workout Plans

* Generates **7-day structured workout routines**
* Includes:

  * Warm-up & cool-down
  * Sets, reps, rest time
  * Morning & evening schedules
* Tailored to:

  * Age, weight, height
  * Fitness level & goals
  * Available equipment (home/gym)

---

### 🥗 Smart Diet Planner

* Creates **custom Indian diet plans**
* Includes:

  * Full-day meal schedule
  * Calories (approx.)
  * Protein-rich suggestions
  * Hydration tips

---

### 🎥 Exercise Tutorials

* Search and watch **YouTube workout tutorials**
* Displays top relevant videos for any exercise

---

### 📊 BMI Calculator & Body Metrics

* Calculates **Body Mass Index (BMI)**
* Displays category:

  * Underweight
  * Normal
  * Overweight
  * Obese

---

### 📈 Progress Tracker

* Track daily weight
* Visualize progress using charts

---

### 📜 History Tracking

* Stores previously generated:

  * Workout plans
  * Diet plans
* Easily accessible in sidebar

---

### 🤖 AI Fitness Coach (Sidebar Chatbot)

* Ask fitness-related questions
* Get instant AI-powered guidance

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **LLM:** Groq (LLaMA 3.1)
* **Framework:** LangChain
* **Search Tool:** DuckDuckGo (ddgs)
* **Data Handling:** Pandas

---

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/your-username/ai-fitness-planner.git

# Navigate to project folder
cd ai-fitness-planner

# Create virtual environment
python -m venv venv
venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file and add:

```
GROQ_API_KEY=your_api_key_here
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```
